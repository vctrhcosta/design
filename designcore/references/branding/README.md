# branding/ — Identidade Visual e Sistemas de Marca

Referências de identidade visual: logos, sistemas de marca, brand books, aplicações em múltiplos formatos, paletas institucionais.

**Contexto de aplicação:** Projetos onde o design precisa ser sistêmico e sobreviver em múltiplos pontos de contato ao longo do tempo.

**Nuance:** Branding bem feito tem uma coerência que vai além do visual — há uma lógica de valores e posicionamento por trás. Referências aqui são mais úteis quando o contexto de negócio é conhecido.
