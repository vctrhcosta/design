# client/ — Referências para Projetos de Clientes

Referências coletadas no contexto de projetos externos para clientes específicos. Organizar em subpastas por cliente quando o volume justificar.

**Contexto de aplicação:** Alta variabilidade — depende do segmento, posicionamento e audiência do cliente.

**Nuance:** Referências aqui são coletadas com um brief específico em mente. Ao analisar, é essencial registrar *para qual cliente/segmento* foram coletadas. Uma referência "certa" para um cliente pode ser completamente errada para outro.

## Estrutura recomendada para projetos grandes

```
client/
  nome-do-cliente/
    refs.md       # contexto + brief resumido
    img1.jpg
    img2.png
```
