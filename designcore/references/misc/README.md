# misc/ — Referências Transversais

Referências que cruzam múltiplas categorias ou que ainda não foram classificadas.

**Uso:** Área de triagem. Imagens aqui devem ser movidas para a pasta correta assim que o contexto de aplicação ficar claro.

**Valor especial:** Referências que genuinamente pertencem a múltiplas categorias revelam *conexões* entre contextos — anotar essas conexões no SOUL.md quando identificadas.
