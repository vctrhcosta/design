# portfolio/ — Portfólios de Designers e Estúdios

Referências de portfólios pessoais, estúdios de design, agências criativas e apresentações de trabalho.

**Contexto de aplicação:** Projetos autorais onde a voz criativa do designer é o produto. Maior liberdade expressiva, menor compromisso com conversão ou clareza funcional imediata.

**Nuance:** Portfólios frequentemente testam o limite do que é possível visualmente. Referências aqui podem ser provocações estéticas que *não* se traduzem diretamente para outros contextos — mas são úteis como expansão de vocabulário.
