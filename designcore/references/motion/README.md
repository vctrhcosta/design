# motion/ — Motion Design e Animação

Referências de motion design: animações de interface, title sequences, brand motion, vídeos conceituais, frames de after effects, Remotion, Lottie.

**Contexto de aplicação:** Projetos com componente temporal — onde o design se desdobra no tempo.

**Nuance:** Motion comunica personalidade de formas que o estático não consegue. Referências aqui são úteis para entender ritmo, easing, e como uma identidade visual "se move" — relevante para onboarding de produto e storytelling de marca.
